### Install and usage:

Install and run:
```bash
pip install -r requirements.txt
python main.py
```

Tested on:
* Windows 11
* Python 3.11.0
* pip 22.3

Test libs versions:
* PyQt5==5.15.7
* Pillow==9.3.0
* pycryptodome==3.15.0
